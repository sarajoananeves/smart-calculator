# Smart Calculator 🧮

A full-stack smart calculator built as a learning project — returning to software engineering after 18 months of maternity leave.

**Frontend**: React + TypeScript + Vite  
**Backend**: Spring Boot 3.5 + Java 17  
**Smart input**: Natural language parser ("7 plus 3", "15% of 80")  
**Testing**: 123 automated tests (Vitest, JUnit, MockMvc, Playwright) — 100% coverage  
**Manual tests**: 10 structured tests for accessibility + keyboard navigation + unreachable service  
**Cross-browser**: Playwright E2E across Chromium, Firefox, WebKit  

Built milestone by milestone. Every commit tells a chapter of the story.
